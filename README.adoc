= SmallRye BeanBag

A simple programmatic bean container. Includes integration for Eclipse SISU as well as a Maven Resolver factory which uses it to wire a `RepositorySystem` instance.

== Instructions

=== Compile and test

To compile and test the project:

[source,bash]
----
mvn verify
----
